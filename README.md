# Portfólio — Mônica Maschio

Site pessoal desenvolvido do zero com HTML, CSS e JavaScript puro, sem frameworks ou bibliotecas externas.

🔗 **[mgamamaschio.github.io](https://mgamamaschio.github.io)**

---

## Sobre o projeto

Este portfólio marca o início da minha transição da logística para o desenvolvimento web. Depois de mais de 10 anos como analista sênior — criando controles, automatizando processos e usando dados para tomar decisões — estou agora aplicando essa mesma mentalidade para construir software.

O site foi desenvolvido com foco em:
- Layout limpo e responsivo (funciona bem no celular e no desktop)
- Código organizado e sem dependências externas
- Carregamento rápido (zero bibliotecas JS, fonte do Google Fonts)

---

## Tecnologias usadas

- HTML5
- CSS3 (Flexbox, Grid, variáveis CSS, media queries)
- JavaScript puro
- Google Fonts (DM Sans + DM Mono)
- GitHub Pages (hospedagem gratuita)

---

## Como rodar localmente

Não é preciso instalar nada. Basta clonar o repositório e abrir o arquivo no navegador:

```bash
git clone https://github.com/mgamamaschio/mgamamaschio.github.io
cd mgamamaschio.github.io
# abra o arquivo index.html no seu navegador
```
